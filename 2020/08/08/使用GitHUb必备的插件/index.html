使用GitHub必备的插件

在这里分享一下关于github的几个特别好用的插件。

一、GitHub Repository Size

​ 最近在GitHub上找项目的时候突然想到：能不能在clone之前就查看到仓库大小呢？在网上查阅了一下，方法不一，其中最方便的应该就是安装Chrome插件了。

​ 该插件所在的仓库里有原作者给出的说明，这里我会把步骤具体展示一下。

​ 点击此处前往该插件所在仓库。

准备工作

​ 在使用该插件之前，我们还需要在GitHub上生成Personal access token。点击此处前往生成token，会出现以下界面。

点击生成token，输入密码之后会出现以下界面。Note处可随意填写，但后面的要全部勾选。然后点击下面的Generate token按钮。

注意！点击Generate token之后，这个token只会明码显示一次！一定要保留下来备用。

安装插件

​ 在将token记录下来之后，我们便可以开始Chrome插件的安装了。首先前往仓库地址将其clone到本地。之后，我们转到chrome扩展程序：chrome://extensions/（也可以从设置里进入）。打开开发者模式，并点击加载已解压的扩展程序。

​ 选择我们clone的仓库的本地地址。完成之后扩展程序里会新增我们添加的程序并会弹出以下两个窗口。

​ 第一个窗口点击确定，然后在第二个窗口处填写我们在上面创建的token，之后点击确定，我们的插件就算是安装完毕了，来看一下效果如何：

此时已经能够看到文件大小了。如果要看文件夹大小，点击那三个小点...即可看到。

​ 该插件可直接在Chrome商店下载，不能浏览的可以自行在网上下载到本地，再按照上述的方法添加到扩展程序中（此插件不需要添加token）。

三、GitZip for github

​ 在使用github的时候，如果你想下载单个文件时该怎么办呢？别担心，这个插件将完美解决这个问题。我们先来看一下效果：
